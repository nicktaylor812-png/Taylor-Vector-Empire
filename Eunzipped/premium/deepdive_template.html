Methodology

TUSG% (True Usage Percentage)

Measures the percentage of team possessions used by a player when on the court.
Formula: (FGA + TOV + (FTA × 0.44)) / ((MP/48) × TeamPace) × 100
Interpretation: Higher values indicate more offensive responsibility

PVR (Possession Value Rating)

Measures the value created per possession used by a player.
Formula: [(PTS + (AST × Multiplier)) / (FGA + TOV + (0.44 × FTA) + AST) - 1.00] × 100
Interpretation: Higher values indicate more efficient production
