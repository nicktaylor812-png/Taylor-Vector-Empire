API Endpoints

Base URL

https://taylor-vector-terminal.replit.app/api/v1

1. Get Player Stats

GET /api/v1/player/{player_name}
curl -H "X-API-Key: YOUR_API_KEY" \ https://taylor-vector-terminal.replit.app/api/v1/player/lebron_james Response: { "player": "LeBron James", "season": "2012-13", "metrics": { "tusg": 31.03, "pvr": 39.21 }, "stats": { "mpg": 37.9, "ppg": 26.8, "apg": 7.3 } }

2. Get Leaderboard

GET /api/v1/leaderboard?metric=tusg&limit=50
curl -H "X-API-Key: YOUR_API_KEY" \ "https://taylor-vector-terminal.replit.app/api/v1/leaderboard?metric=tusg&limit=10" Response: { "metric": "tusg", "count": 10, "leaderboard": [ { "rank": 1, "player": "Russell Westbrook", "season": "2016-17", "tusg": 48.1, "pvr": 25.11 } ] }

3. Get Live Betting Edges

GET /api/v1/edges?min_confidence=65
curl -H "X-API-Key: YOUR_API_KEY" \ "https://taylor-vector-terminal.replit.app/api/v1/edges?min_confidence=70" Response: { "count": 5, "edges": [ { "game": "Lakers vs Warriors", "pick": "Lakers -3.5", "edge": 72.5, "home_tusg": 45.2, "away_tusg": 38.7 } ] }

Embeddable Widgets

Integrate beautiful, responsive widgets directly into your site using simple iframe embeds. All widgets support custom branding and track analytics for revenue sharing.

1. Leaderboard Widget

Display top players ranked by TUSG% or PVR with interactive sorting.

<iframe src="https://taylor-vector-terminal.replit.app/partnerships/widget/leaderboard?partner=YOUR_API_KEY&color=667eea&limit=10" width="100%" height="600" frameborder="0"> </iframe>

2. Comparison Widget

Let users compare any two players across all metrics.

<iframe src="https://taylor-vector-terminal.replit.app/partnerships/widget/comparison?partner=YOUR_API_KEY&color=f5576c" width="100%" height="600" frameborder="0"> </iframe>

3. Live Edge Feed Widget

Display real-time betting edges with auto-refresh.

<iframe src="https://taylor-vector-terminal.replit.app/partnerships/widget/edge-feed?partner=YOUR_API_KEY&color=2ecc71&min_edge=65" width="100%" height="600" frameborder="0"> </iframe>

Customization Parameters

  • partner - Your API key (required for revenue tracking)
  • color - Primary color (hex without #, e.g., "667eea")
  • logo - URL to your logo (URL-encoded)
  • limit - Number of items to display (leaderboard only)
  • min_edge - Minimum confidence % (edge feed only)

Webhook System

Receive real-time notifications when data updates occur. Perfect for keeping your platform synchronized with the latest metrics.

Setting Up Webhooks

Provide your webhook endpoint when you sign up. We'll POST updates to your URL with the following structure:

POST https://your-site.com/webhooks/tusg-update Headers: Content-Type: application/json X-Webhook-Signature: YOUR_WEBHOOK_SECRET Payload: { "event": "data_update", "timestamp": "2025-11-19T12:34:56Z", "data": { "updated_players": 150, "new_edges": 5 }, "source": "TAYLOR VECTOR TERMINAL" }

Event Types

  • data_update - Leaderboard data has been updated
  • new_edge - High-confidence betting edge detected
  • player_update - Individual player stats updated
